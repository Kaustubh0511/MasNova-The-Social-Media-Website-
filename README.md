# MasNova-The-Social-Media-Website-# Asocial Media

Asocial Media is a unique social networking platform that aims to provide a different social media experience to its users. Unlike traditional social media platforms that focus on connecting people and fostering communication, Asocial Media encourages users to embrace solitude and disconnection from others. This README file provides an overview of the project, its features, and instructions for setting up and running the application locally.

## Features
- **Isolation Mode**: Asocial Media creates a personalized experience by minimizing all forms of social interactions. Users can customize their profiles to be completely isolated from others.
- **Limited Sharing**: The platform restricts the ability to share posts, images, or other media with other users. Content remains private and solely accessible to the creator.
- **No Comments or Likes**: Asocial Media eliminates comments and likes on posts to discourage engagement and maintain a peaceful environment.
- **Dark Mode**: The application provides a dark mode interface to enhance the solitary experience.

## Requirements
To run Asocial Media locally, ensure you have the following dependencies installed:
- Node.js (v12 or higher)
- npm (v6 or higher) or yarn (v1 or higher)
- MongoDB

## Installation
1. Clone the repository to your local machine:
```shell
git clone https://github.com/username/asocial-media.git
```
2. Navigate to the project directory:
```shell
cd asocial-media
```
3. Install the dependencies:
```shell
npm install
```
or
```shell
yarn install
```
4. Configure the application by creating a `.env` file in the root directory. Refer to the `.env.example` file for the required environment variables.
5. Start the application:
```shell
npm start
```
or
```shell
yarn start
```

## Usage
1. Open your web browser and visit `http://localhost:3000` to access the Asocial Media application.
2. Sign up with a new account or log in if you already have one.
3. Customize your profile settings to enable isolation mode and adjust other preferences.
4. Start creating and sharing content with the peace of mind that it will remain private and disconnected from others.
